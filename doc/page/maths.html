MATHS ( ... )

Maths est une fonction du module purplesand qui permet de transformer du type string en type number de manière fiable sans utiliser l'évaluation javascript.
Sommaire

Syntaxe

maths('string').methode();

La méthode maths ne prend qu'un seul argument de type string.

Méthodes

La fonction maths contient différentes méthodes de conversion. La fontion maths n'utilise pas la fonction eval() de javascript, maths convertit elle même la chaine en nombre/calcul de manière robuste et sécurisée.

toNumber

maths('56.8').toNumber();°r /* retourne 56.8 de type number */

Cette méthode retourne le nombre correspondant à l'argument string.

calc

maths('8.6+7.6^(2-1)').calc();°r /* retourne 16.2 de type number */

Cette méthode retourne le nombre correspondant au calcul demandé dans l'argument string.

calcZone

maths('le résultat de 7 + 89 est {7+89}').calcZone();°r /* retourne 'le résultat de 7 + 89 est 96' */

Cette méthode retourne les résultats des calculs contenus dans les espaces de calculs déterminés par des accolades, au sein de la chaine.
