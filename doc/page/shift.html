SHIFT | SH (global)

Shift est une méthode de la fonction pps qui permet préciser la méthode start.
Sommaire

Syntaxe

shift: 0

La méthode shift ne prend qu'un seul argument de type number.

Il s'agit d'un entier qui vient s'ajouter à la méthode start.

pps({ °st$: { °st$: 'elements'°et },°et °ststart: 4,°r shift: -1°r /* ici le starter est le troisième élément (4 - 1) */°et });
