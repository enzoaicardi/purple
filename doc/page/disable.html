DISABLE

Disable est une méthode de la fonction pps qui permet désactiver une portion de queue.
Sommaire

Syntaxe

disable: true | false

La méthode disable ne prend qu'un seul argument de type boolean. Si ce dernier vaut true alors la portion de queue dans laquelle disable est utilisée sera désactivée.

pps({°st disable: true°r /* cette queue est désactivée */°r /* on le la lit pas, on passe directement à la suivante */ °et},°r {°st css: ...°r /* cette queue est active */ °et});

Si disable est spécifiée dans la queue principale alors c'est tout la queue qui sera désactivée. Mais si disable est spécifié dans une portion comme if ou foreach alors seule la portion sera désactivée.

pps({°st foreach: {°st /* cette queue foreach est désactivée */°r disable: true °et}°r /* mais la queue est active */°r css: ... °et},°r {°st if: { condition: ... °st /* cette portion if est désactivée */°r disable: true °et}°r /* mais la queue est active */°r css: ...°r °et});

Disable peut s'avérer utile pour le débogage.
