FREQUENCE | FREQ (global)

Frequence est une méthode de la fonction pps qui permet de déterminer la frequence de sélection des éléments.
Sommaire

Syntaxe

frequence: 1 | 'string'

La méthode frequence ne prend qu'un seul argument pouvant être de différents types et qui détermine la frequence de sélection des éléments.

Toutes les méthodes non-globales, c'est à dire qui font références aux éléments plutôt qu'au contexte, sont affectées.

Arguments

La méthode frequence peut prendre des arguments de plusieurs types différents.

Type Number

Avec une fréquence de 2, un élément sur deux sera affecté par les méthodes non-globales.

pps({ °st$: { °st$: 'elements',°r°et },°et °stfrequence: 2°et });

La méthode frequence ne doit pas être confondue avec le paramètre freq du sélecteur.

Type Percentage

La fréquence peut être indiquée sous la forme d'un pourcentage, ce dernier renvoie un index qui servira de nombre de référence, voir Min et Max du sélecteur.

pps({ °st$: { °st$: 'elements',°r°et },°et °stfrequence: '20%'°r /* pour une liste de 10 éléments, un élément sur 2 sera sélectionné */°et });
