LISTE DES PARAMETRES

L'objet params qui peut être renvoyé dans des fonctions comme foreach, function ou event contient différents paramètres.
Sommaire

Les différents paramètres correspondent aux valeurs des différentes méthodes globales.

elements

params.elements

Renvoie le tableau de tout les HTMLElement désignés par le sélecteur, ce paramètre n'est donc pas affecté par start, mais est affecté par les paramètres du sélecteur (min, max, etc.).

[element_1, element_2, ... ]

index

params.index

N'est disponible que pour les fonctions qui font appel à une boucle comme foreach. L'index n'est donc pas disponible pour function.

Il renvoie un entier correspondant à l'index de l'élément suivant son ordre d'affectation. Cet ordre est changé par la propagation. Pour une propagation de type 'center' les deux éléments déchenchés simultanéments auront le même index. Cela permet de mieux contrôler les effets liées aux espaces de variables dans les méthodes d'animation du DOM.

1, 2, 3 ... /* l'index commence toujours par 1 */

Le mode de propagation reverse fait exeption, le premier index est 0, tandis que les autres index sont opposés (1 / -1, 2 / -2, etc...).

Le véritable index de l'élément dans le tableau des éléments correspond à :

params.elements.indexOf(item);

start

params.start

Renvoie la valeur de la dernière méthode start.

frequence

params.frequence

Renvoie la valeur de la dernière méthode frequence.

propagation

params.propagation

Renvoie la valeur de la dernière méthode propagation.

interval

params.interval

Renvoie la valeur de la dernière méthode interval.

wait

params.wait

Renvoie la valeur de la dernière méthode wait.

reset

params.reset

Renvoie la valeur de la dernière méthode reset.
