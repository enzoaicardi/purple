PPS ( ... )

pps ( ... ) est la fonction principale du module PurpleSand, c'est elle qui permet de définir son fonctionnement. Toutes les méthodes du module PurpleSand ne sont utilisables qu'à l'intérieur de cette fonction.
Sommaire

Syntaxe

pps(argument);

La fonction pps est la fonction primaire du module PurpleSand, elle ne prend qu'un seul argument pouvant être de plusieurs types.

PurpleSand s'organise selon un système de queues, ce dernier est expliqué dans le reste de la documentation.

Arguments

La fonction pps ( ... ) peut prendre des arguments de plusieurs types différents.

Type String

Avec un seul argument de type string, la fonction pps renvoie le premier élément trouvé grace à la méthode querySelector.

pps('string'); /* => document.querySelector('string'); */

La fonction est donc chaînable (uniquement dans ce cas).

pps('string').classList;°r /* => document.querySelector('string').classList; */

Type Object

Avec un seul argument de type object, il est possible de spécifier des pairs {clé: valeur} pour exploiter toutes les méthodes du module PurpleSand.

pps({ °stclé: 'valeur'°et });

Multiple Objects

Avec plusieurs argument de type object, il est possible de spécifier des pairs {clé: valeur} pour exploiter toutes les méthodes du module PurpleSand.

pps( °st{ °stclé: 'valeur'°et },°et °st{ °stclé: 'valeur'°et }°et );
