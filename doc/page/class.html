CLASS | CL

Class est une méthode de la fonction pps qui permet de d'attribuer des classes à des éléments.
Sommaire

Syntaxe

class: {°st add: 'class_name',°r remove: 'class_name',°r replace: {°st old: 'old_class_name',°r new: 'new_class_name' °et},°r contains: 'class_name',°r statement: true °et}

La méthode class permet d'ajouter, de supprimer ou de remplacer le ou les classes des éléments sélectionnés.

Paramètres

La méthode class comprend plusieurs paramètres permettant de faire plusieurs opérations.

Add

Add permet d'ajouter la classe renseignée aux objets sélectionnés via le sélecteur.

pps({ °stclass: {°st add: 'class-1' °et}°et });

Remove

Remove permet de retirer la classe renseignée aux objets sélectionnés.

pps({ °stclass: {°st remove: 'class-2' °et}°et });

Replace

Replace permet de replacer une classe par une autre pour les objets sélectionnés.

pps({ °stclass: {°st replace: {°st old: 'old-class',°r new: 'new-class' °et} °et}°et });

Old

Old désigne la classe à remplacer.

pps({ °stclass: {°st replace: {°st old: 'old-class' °et} °et}°et });

New

New désigne la classe qui remplacera old.

pps({ °stclass: {°st replace: {°st new: 'new-class' °et} °et}°et });

Contains

Contains est un paramètre conditionnel, si la classe renseignée est présente alors les autres paramètres seront actifs, sinon aucun changement ne sera effectué.

pps({ °stclass: {°st add: 'class-to-add',°r contains: 'class-contained' °et}°et });

Dans l'exemple ci-dessus : si l'élément possède la classe 'class-contained' alors on ajoute à l'élément la classe 'class-to-add', sinon on ne fait rien.

Statement

Le comportement de contains peut être changé grâce au paramètre statement. Par défaut à true c'est un booléen qui détermine si les changements son effectués quand contains vaut true ou false.

pps({ °stclass: {°st add: 'class-to-add',°r contains: 'class-contained',°r statement: false °et}°et });

Dans l'exemple ci-dessus : si l'élément ne possède pas la classe 'class-contained' alors on ajoute à l'élément la classe 'class-to-add', sinon on ne fait rien.
