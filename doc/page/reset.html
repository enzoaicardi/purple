RESET | RS (global)

Reset est une méthode de la fonction pps qui permet de les valeurs par défaut des méthodes globales.
Sommaire

Syntaxe

reset: true | false

La méthode reset ne prend qu'un seul argument de type boolean, si faux alors les valeurs des méthodes globales seront concervées d'une queue à l'autre.

Toutes les méthodes globales, c'est à dire celles qui font références au contexte, sont affectées, à l'exception de la méthode reset et du sélecteur.

Exemple d'utilisation

pps( °st{ °st$: 'elements',°r reset: false,°r interval: 1000,°r css: [1000,{backgroundColor: 'purple'}] °et },°r { °st /* les paramètres de la queue précédente sont concervés */°r /* interval = 1000ms */°r css: [1000,{backgroundColor: 'gray'}] °et }°et );

Dans l'exemple suivant la deuxième méthode css prendra le même interval que la première.
