Include est une méthode de la fonction pps qui permet d'inclure un fichier dans un élément.
Sommaire

Syntaxe

include: 'file-URL'

Include permet d'inclure un fichier dans un élément à partir de son URL.

Argument

La méthode include ne prend qu'un seul argument de type string ou array.

String

Il s'agit de l'URL du fichier que l'on souhaite inclure dans les éléments sélectionnés via le sélecteur. Le contenu des éléments est alors remplacé par le nouveau contenu.

pps({°st $: 'elements',°r include: 'file-URL.html' °et});

Array

Il est possible de spécifier un tableau comme argument de la manière qui suit :

include: ['URL', {options}, (res)=>{function}]°r /* ou ... */°r include: ['URL', (res)=>{function}]

URL

Il s'agit du chemin pour accéder au fichier.

Options

Il s'agit des options de l'API Fetch.

Fonction

Spécifier une fonction permet de modifier le résultat de la requête. La fontion doit contenir un seul argument qui correspond au résultat de la requête. Une fois le résultat modifié il est impératif de le retourner avec l'instruction return.

include: ['URL', (res)=>{return res.replace('something', 'other');}]
